Static Portfolio Webpage (HTML/CSS)
Project Description

This is a simple static portfolio website created using HTML and CSS.
It showcases personal information such as an introduction, skills, projects, and contact details.
This project is part of the Computer Science Fundamentals & Career Pathways (ETCCCP105) assignment, demonstrating the use of VS Code, Git, and GitHub for building and hosting a web project.

Installation & Usage Instructions﻿

1. git clone https://github.com/Sarthak-13-code/Static-Portfolio-Webpage-HTML-CSS.git

2. Open the Project

Open the folder in VS Code or any code editor.

3. Run the Website

Simply open the file:

index2.html

in any web browser (Chrome, Edge, Firefox, etc.).

Features of the Project
✔ 1. Clean & Simple Homepage

The homepage gives a neat introduction to the portfolio owner.

✔ 2. About Me Section

Includes basic personal details, goals, and background.

✔ 3. Skills Section

Shows your technical skills in a structured manner.

✔ 4. Project Showcase

Displays your past or sample projects with HTML/CSS.

✔ 5. Additional Webpage

index2.html provides extended content or extra page navigation.

✔ 6. Custom Styling

Uses style.css for layout, colors, typography, and structure.

✔ 7. Fully Static & Lightweight

Runs directly in the browser without requiring any backend or installation.
